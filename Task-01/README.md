# House Price Prediction using Linear Regression

## Project Overview
This project predicts house prices using a Linear Regression model based on housing features.

## Dataset
House Prices Dataset (train.csv)

## Features Used
- GrLivArea
- BedroomAbvGr
- FullBath

## Target Variable
- SalePrice

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Matplotlib

## Workflow
1. Load Dataset
2. Data Exploration
3. Feature Selection
4. Train-Test Split
5. Train Linear Regression Model
6. Make Predictions
7. Evaluate Model
8. Visualize Results

## Results
- Dataset Shape: (1460, 81)
- Training Data Shape: (1168, 3)
- Testing Data Shape: (292, 3)
- R² Score: 0.63
